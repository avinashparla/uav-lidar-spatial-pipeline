# Research Foundations & Academic Pedigree

The algorithms in this repository are an extension of over half a decade of research into high-altitude environments and geodetic spatial analysis. 

### **1. Geodetic Mass Balance (IIT Bombay)**
* **Focus:** Utilizing the "Geodetic Method" to estimate ice thickness and surface mass balance in the Chandra Basin, Himalayas.
* **Significance:** This work required the alignment of multi-temporal satellite datasets and DGPS field surveys, directly informing the co-registration logic used in this LiDAR pipeline.

### **2. Feature-Oriented PCA (AGU 2020)**
* **Presentation:** "Glacier Snow Area Mapping Using Feature Oriented Principle Component Analysis."
* **Significance:** Demonstrated advanced statistical decomposition of spatial data to isolate specific terrain features—a principle now applied to noise filtering and classification in my Python pipelines.

### **3. Remote Sensing & GIS Lab (UBC)**
* **Role:** Doctoral Researcher / Teaching Assistant.
* **Focus:** Mapping high-altitude environments using advanced GIS and Remote Sensing methodologies. This tenure refined my ability to architect automated workflows for massive spatial datasets.

### **4. Global Collaboration (WGMS Zurich)**
* **Role:** Guest Researcher.
* **Focus:** Training in international standards for glacier monitoring and data standardization, ensuring all code produced meets global geodetic benchmarks.
